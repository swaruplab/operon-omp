Getting started

Installing Operon on macOS

Download the app, drag it into Applications, and walk through the first-launch setup wizard. Covers both Apple Silicon and Intel builds, plus automatic dependency installation.

Remote & HPC

Connecting to a Remote Server

Set up an SSH connection profile to a remote server or HPC cluster. Covers hostname, port, SSH keys, password authentication, and Duo / MFA logins.

Remote & HPC

Installing Claude on a Remote Server

Install Claude Code on your HPC login node so that agent and plan modes run on compute resources instead of your laptop. Shell detection, curl install, and verification.

Analysis

Single-cell RNA-seq with Scanpy

Run a basic scRNA-seq analysis end-to-end using Scanpy inside Operon — loading data, QC filtering, normalization, clustering, and UMAP — all driven by the chat panel.

Cell Segmentation with CellPose

Use CellPose within Operon to segment cells from microscopy images. Model selection, input preparation, running segmentation, and inspecting the output masks.
