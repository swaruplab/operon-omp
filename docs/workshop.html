A one-day, hands-on Operon workshop for biologists and computational researchers. Install Operon, connect it to an HPC, and use Claude as a full research collaborator — from raw data to finished figures. No prior experience with AI agents assumed.

About the workshop

What you'll leave with

By the end of the day you'll have used Operon to run a real bulk or single-cell pipeline end-to-end, connected it over SSH to a remote compute node, and generated a publication-ready report from your session. You'll also know how to import your lab's existing pipelines as reusable protocols, when to reach for Agent vs. Plan vs. Ask mode, and how to keep Claude honest with PubMed-grounded citations.

Open to graduate students, postdocs, PIs, and research staff working in genomics, single-cell, imaging, or adjacent fields. Seats are limited.

Rough agenda

  1. Morning · Installation hands-on
    Install Operon on your own laptop — macOS, Windows, or Linux. First-launch setup, dependency checks, Anthropic API key, tour of the workspace, and a walk-through of Agent vs. Plan vs. Ask vs. Report. Everyone leaves lunch with a working install.
  2. Lunch
    Catered · Q&A with the Operon team.
  3. Afternoon · Running analysis locally
    Drop in a FASTQ or h5ad on your own machine. Watch Claude run QC, alignment/normalization, and clustering. Iterate in chat, review the diff, accept changes.
  4. Late afternoon · Running analysis on the HPC
    Connect over SSH, run the same pipeline inside a tmux session on a compute node. Resume the session later from a fresh browser. Bring your cluster credentials.
  5. Wrap
    Common gotchas · where to ask questions · where Operon is going next.

What to bring

  • A laptop (macOS, Windows, or Linux)
  • Your SSH credentials for an HPC cluster (optional but recommended — bring your institution's login node hostname)
  • A dataset you actually want to analyze — FASTQ, BAM, h5ad, count matrix, microscopy image, anything
  • An Anthropic API key, or we'll get you set up with one on the day

Prerequisites

None. If you've ever opened a terminal you'll be fine. The workshop is designed so an experimental biologist who writes no code and a computational postdoc who writes pipelines every day both leave with something useful.

Taught by the Swarup Lab

Operon is developed at the Swarup Lab at UC Irvine — a neurogenomics group running single-cell, spatial, and epigenomic experiments in Alzheimer's disease. The workshop is led by the team that builds Operon day-to-day.
